Acerca de Salem TV

¿Cómo usar la app?

Selecciona un canal de la lista principal para comenzar a reproducir. Si el canal tiene múltiples opciones, podrás elegir la que mejor te convenga.

¿La app consume muchos datos?

Esto depende de la calidad del stream. Se recomienda usar Wi-Fi para una mejor experiencia.

¿Puedo usarla en iOS/Android?

Sí, este proyecto está optimizado para funcionar en cualquier dispositivo.

¿Publicidad molesta?

Para usuarios IOS se recomienda descargar Adblock for Safari o Navegador Brave en la AppStore.

Para usuarios Android se recomienda descargar AdGuard o Navegador Brave en la PlayStore.

Con estos complementos se busca mejorar la experiencia de usuario, eliminando de manera segura la publicidad de terceros.

Ticket/Ayuda

Para soporte de caida de canal, petición de canal, errores, donaciones, etc. Únicamente via YAPE.

Nota

No alojamos ni transmitimos señales propias. Las transmisiones provienen de sitios externos, incluyendo la totalidad de los anuncios.
