Acerca de Salem TV

¿Cómo usar la app?

Selecciona cualquier canal de la lista principal para comenzar a reproducir. Si un canal tiene múltiples opciones, podrás elegir la que mejor te convenga.

Preguntas frecuentes

¿La app consume muchos datos?
Esto depende de la calidad del stream. Se recomienda usar Wi-Fi para una mejor experiencia.

¿Puedo usarla en iOS/Android?
Sí, está optimizada para funcionar en cualquier dispositivo móvil, si observas algún problema no dudes en avisarnos.

¿Si un canal no está disponible?

Si las señales se caen, presentan problemas o no se encuentra un canal, avisanos para comenzar a trabajar en ello y traer el contenido que se necesita.

¿Publicidad molesta?

Para usuario IOS se recomienda descargar Adblock for Safari o Navegador Brave.

Para usuario Android se recomienda descargar AdGuard o Navegador Brave.

Los anuncios son externos y se recomienda descargar para una mejor experiencia de usuario.
